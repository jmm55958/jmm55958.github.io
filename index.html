This Site - Github - Pages

This website was built using Github pages!

Github pages allows you to host your repository, one per account! This essentially creates your portfolio.

It is required to have a Github account (free for students), and there can only be one site hosted per account.

Steps

  • Create a new repository named *your account name*.github.io
  • Populate this repository, needs an index.html
  • That's it! In repository settings, there should be a section on Github pages where it shows the options.

Any changes need to be pushed to the master branch of the repository, and Github will handle the rest!

4310 - Rich Media Final

For my final in rich media production, I decided to delve into a newer field for the new media app development, persistent data storage.

I created an ios application that acted as a populated list. Upon launch, user is greeted with a splash screen, something extra for user experience.
Then users can choose to review their list or add another item.
The add button takes users to a scrollable screen, another extra. Upon submitting this form, the new item will be added to the list.
When the app is closed and the re-launched, the data in the list remains constant. I did this through the use of persistent data.

This app contains many features that could not be covered in this small space so check out my video where I go into depth.
